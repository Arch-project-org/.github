# Política de Segurança

## Versões Suportadas
As atualizações de segurança são aplicadas continuamente à versão principal em produção.

## Reportar uma Vulnerabilidade
**Não abra uma Issue pública para reportar falhas de segurança.** 

Para reportar uma vulnerabilidade confidencialmente, envie um e-mail para **security@arch.ia.br** detalhando:
1. O tipo de vulnerabilidade.
2. Os passos para reproduzir o problema.
3. O impacto potencial percebido.

A nossa equipa confirmará a receção no prazo de 48 horas e fornecerá atualizações sobre a correção.
